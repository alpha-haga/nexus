# agent/assignment

## 責務

**案件割当の管理**。

- 代理店への案件アサイン
- 紹介元の記録
- 割当履歴の管理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| agent/contract | 参照 | 契約有効性の確認 |
| agent/commission | 連携 | 割当→報酬計算 |
| gojo | 参照 | 互助会案件の紹介元 |
| funeral | 参照 | 葬祭案件の請負元 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 互助会契約の作成 | gojo の責務 |
| 葬祭案件の作成 | funeral の責務 |
| 報酬計算 | commission の責務 |
