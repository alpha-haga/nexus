# agent/commission

## 責務

**支給・報酬の管理**。

- 報酬計算
- 支給明細の作成
- 報酬履歴の管理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| agent/contract | 参照 | 手数料率の取得 |
| agent/assignment | 参照 | 実績の取得 |
| payment | イベント送信 | 支払 Fact |
| accounting | イベント送信 | 会計 Fact |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| **会計仕訳の作成** | accounting の責務 |
| **支払処理の実行** | payment の責務 |
| 代理店契約の管理 | contract の責務 |
| 案件割当の管理 | assignment の責務 |

## 会計との境界

```
┌─────────────────┐         ┌─────────────────┐
│agent/commission │         │   accounting    │
│  ───────────    │         │  ───────────    │
│  報酬計算       │────────▶│  費用仕訳       │
│  支給明細作成   │         │  未払金仕訳     │
│                 │         │                 │
│  ※Factのみ      │         │  ※仕訳を作る    │
└─────────────────┘         └─────────────────┘
```

commission は「いくら支給する」という **事実** を記録。
その事実を **仕訳** にするのは accounting の責務。
