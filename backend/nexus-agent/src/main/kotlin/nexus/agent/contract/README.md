# agent/contract

## 責務

**業務委託契約の管理**。

- 代理店契約の作成・管理
- 契約条件（手数料率等）の管理
- 契約ステータス管理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| agent/commission | 連携 | 報酬計算の基準 |
| agent/assignment | 連携 | 割当可否の判定 |
| gojo | 参照される | 紹介代理店として |
| funeral | 参照される | 請負代理店として |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 互助会契約の管理 | gojo の責務 |
| 葬祭案件の管理 | funeral の責務 |
| 報酬支払処理 | commission → payment の責務 |
