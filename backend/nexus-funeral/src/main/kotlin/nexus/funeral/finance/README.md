# funeral/finance

## 責務

**葬祭の債権・入金・預り金管理**。

**※会計ではない**。金銭の事実（Fact）を管理する。

- 請求書発行
- 入金消込
- 預り金管理
- 債権管理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| funeral/operation | 連携 | 施行完了→請求 |
| payment | イベント送信 | 入金 Fact を送信 |
| accounting | イベント送信 | 会計 Fact を送信 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| **会計仕訳の作成** | accounting の責務 |
| **勘定科目の管理** | accounting の責務 |
| **借方・貸方の記録** | accounting の責務 |
| **会計ルールの実装** | accounting の責務 |
| 顧客情報の更新 | identity の責務 |
| 施行管理 | operation の責務 |

## 重要な境界

```
┌─────────────────┐         ┌─────────────────┐
│ funeral/finance │         │   accounting    │
│  ───────────    │         │  ───────────    │
│  請求書発行     │────────▶│  売上仕訳       │
│  入金記録       │────────▶│  入金仕訳       │
│  預り金記録     │────────▶│  預り金仕訳     │
│                 │         │                 │
│  ※Factのみ      │         │  ※仕訳を作る    │
└─────────────────┘         └─────────────────┘
```

finance は「いくら請求した」「いくら入金された」という **事実** を記録。
その事実を **仕訳** にするのは accounting の責務。
