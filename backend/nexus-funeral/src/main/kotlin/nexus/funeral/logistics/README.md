# funeral/logistics

## 責務

**運行・車両管理**。

- 霊柩車・搬送車両の管理
- 運行スケジュール
- ドライバー配置
- 外注車両の手配

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| funeral/operation | 連携 | 施行に伴う搬送依頼 |
| funeral/finance | 連携 | 運行費用の連携 |
| agent | 参照 | 外注業者の参照 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 施行管理 | operation の責務 |
| 請求処理 | finance の責務 |
| 外注契約管理 | agent の責務 |
