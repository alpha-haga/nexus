# gojo/change

## 責務

**互助会契約の変更処理**。

- プラン変更
- 名義変更
- 受益者変更
- 解約処理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| gojo/contract | 呼び出し先 | 契約情報の更新 |
| gojo/task | 連携 | 変更承認ワークフロー |
| identity | 参照 | 名義人・受益者の確認 |
| payment | イベント送信 | 解約返戻金の Fact |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 顧客情報の更新 | identity の責務 |
| 新規加入処理 | enrollment の責務 |
| 返戻金の支払処理 | payment の責務 |
