# gojo/contract

## 責務

**互助会契約・権利の管理**。

- 契約の作成・保持
- 契約ステータスの管理
- 権利（満期到達、使用可能等）の判定

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| gojo/enrollment | 呼び出し元 | 加入処理から契約作成 |
| gojo/change | 呼び出し元 | 変更処理から契約更新 |
| funeral | 参照される | 葬祭で契約を利用 |
| bridal | 参照される | 冠婚で契約を利用 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 入金処理の実行 | payment の責務 |
| 顧客情報の更新 | identity の責務 |
| 加入審査ロジック | enrollment の責務 |
