# gojo/enrollment

## 責務

**互助会への加入処理**。

- 加入申込の受付
- 審査・承認フロー
- 契約作成の起点

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| gojo/contract | 呼び出し先 | 承認後に契約作成 |
| gojo/task | 連携 | 承認ワークフロー |
| identity | 参照 | 申込者の確認 |
| agent | 参照 | 紹介代理店の記録 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 会計仕訳の作成 | accounting の責務 |
| 顧客情報の作成・更新 | identity の責務 |
| 契約変更処理 | change の責務 |
| 入金処理 | payment の責務 |
