# gojo/master

## 責務

**互助会マスタデータの管理**。

- プランマスタ
- 掛金テーブル
- 商品マスタ

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| gojo/contract | 参照される | 契約時にプラン参照 |
| gojo/enrollment | 参照される | 加入時に商品参照 |
| gojo/change | 参照される | 変更時にプラン参照 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 契約データの管理 | contract の責務 |
| 業務ロジックの実装 | マスタ管理に徹する |
| 会計勘定科目の管理 | accounting の責務 |
| 顧客情報の管理 | identity の責務 |
