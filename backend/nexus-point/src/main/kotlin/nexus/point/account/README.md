# point/account

## 責務

**ポイントアカウントの管理**。

- ポイント残高管理
- ポイント付与・使用
- 取引履歴管理
- 有効期限管理

## 他ドメインとの関係

| ドメイン | 関係 | 説明 |
|---------|------|------|
| identity | 参照 | 所有者の PersonId |
| gojo | 連携 | 互助会契約でのポイント付与 |
| funeral | 連携 | 葬祭サービスでのポイント利用 |
| bridal | 連携 | 冠婚サービスでのポイント利用 |

## 禁止事項

| 禁止 | 理由 |
|------|------|
| 人物情報の更新 | identity の責務 |
| 業務ロジックの実装 | ポイント管理に徹する |
| 会計仕訳の作成 | accounting の責務 |
